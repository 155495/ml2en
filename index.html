ml2en (Python, Javascript, PHP)

An algorithm tp transliterate Malayalam script to Roman / Latin characters (commonly 'Manglish') with relative phonetic fairness

GitHub page Download ZIP

Paste Malayalam input

Romanized ('Manglish') output

Usage

The algorithm's available in three different languages, Python, Javascript, and PHP.

Python

from ml2en import ml2en

converter = ml2en()
result = converter.transliterate(ml_str)

Javascript

<script src="ml2en.js"></script>
<script>
	var result = ml2en.transliterate(ml_str);
</script>

PHP

<?php
	require 'ml2en.php';

	$result = ml2en::transliterate($ml_str);
?>
